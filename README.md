# The Great Books Semantic Engine

Standalone static prototype for a comparative philosophy landing app and semantic search demo.

What is included:

- `index.html` - landing page and live query lab
- `styles.css` - full responsive visual system
- `data.js` - curated corpus, concept lattice, prompts, and product copy
- `main.js` - semantic ranking, counterpart pairing, and graph rendering

How to use it:

1. Open `index.html` directly in a browser, or serve this folder with any static server.
2. Use the preset prompts or type your own research question.
3. Change the primary corpus and result depth to inspect different pairings.

Prototype note:

- This build simulates multilingual embedding behavior with hand-tuned concept vectors.
- It is intended as a working front-end model of the product experience, not the final backend implementation.
